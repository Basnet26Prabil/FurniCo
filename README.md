# FurniCo

Furniture e-commerce platform - CS5054NP coursework at Islington College.

## Tech Stack

- Java 17, Apache Tomcat 10.1, MySQL (XAMPP)
- JSP + JSTL + EL, BCrypt for password hashing
- MVC architecture with Service layer

## Setup

1. Clone this repo
2. Import into Eclipse: File -> Import -> Existing Projects into Workspace
3. Make sure target runtime is Apache Tomcat v10.1
4. Start XAMPP -> Apache + MySQL
5. Open phpMyAdmin -> Import sql/furnico.sql
6. Right-click project -> Run As -> Run on Server
