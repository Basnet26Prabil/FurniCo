#FurniCo
Furniture e-commerce platform - CS5054NI coursework at Islington College.
##Tech Stack
Java 17, Apache Tomcat 10.1, MySQL (XAMPP)
JSP + JSTL + EL, BCrypt for password hashing
MVC architecture with Service layer
##Setup
Clone this repo
Import into Eclipse -> Import -> Existing Projects into Workspace
Make sure target runtime is Apache Tomcat v10.1
Start XAMPP -> Apache + MySQL
Open phpMyAdmin -> Import sql/furnico.sql
Right-click project -> Run As -> Run on Server
